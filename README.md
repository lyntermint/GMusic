## Local development

### Local project

Clone the fork:
```bash
git clone https://github.com/lyntermint/GMusic
```

### Build

Run the `gradlew clean build` command.

The final `GMusic-x.x-x.jar` file will be in the [`build/libs`](./build/libs) folder.
